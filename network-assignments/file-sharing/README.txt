Compile & Execute
- Makefile has been provided in the project directory, simplely type "make" to compile the program, "make clean" to clean things up
- Program usage: ./rfss <port num> 

String Length Assumptions
- All file names is less than 50 characters
- All valid path to a file is less than 100 characters

Hostname Format 
- Valid hostname: xinu<xx>.cs.purdue.edu 
- You may enter xinu<xx>, the program should be able to concatenate "cs.purdue.edu" to user input
